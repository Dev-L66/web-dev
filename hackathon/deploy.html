Media tags

Audio tag

Video tag



It uses the src attribute to specify the path to the video file.

IFRAME TAG for Youtube


It is used to embed a YouTube video.

Deploymenton Netlify

    What you need?
  • A Netlify account.
  • Your HTML file(s).
  • Optionally GitHub (or GitLab / Bitbucket) to host your code.

Steps to deploy a simple HTML page on Netlify

  1. Create a new Netlify project.
  2. Upload your HTML file(s) to the project.
  3. Click Deploy.
  4. Done!
