Forms in HTML

Input element


This is the most common form tag. It's a self-closing tag that creates an input field. Its behavior changes based on its type attribute. The most basic type is text.

label


It's a tag specifically designed to be the title for a form field.

id and for attributes


To create a link between the input and the label, you can use the id and for attributes.

form tag






We wrap all our form fields in a form tag. This tag acts as the main container that tells the browser, everything inside here is part of one single submission. input with type="submit" is used to create a submit button.

Radio button


The radio button is a type of input element that allows the user to select one option from a set of options.

Checkbox


Checkbox is a type of input element that allows the user to select multiple options from a set of options

Button


It's a container tag, meaning it has an opening and closing tag. This allows you to put other HTML elements inside it.

Textarea


Color


It's a type of input element that allows the user to select a color.

Select


It is used to create a drop-down list of options in forms
