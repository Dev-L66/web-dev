Nested lists

A nested list is a list inside another list item. You can use either ordered ol or unordered ul lists, and nest as many levels as needed (though too many can get confusing).

Tables

The table tag is used to create a table. The th tag is used to create a table header. The tr tag is used to create a table row. The td tag is used to create a table data. The thead tag is used to create a table header. The tbody tag is used to create a table body. The tfoot tag is used to create a table footer.

  • colspan: The colspan attribute in a table cell td or th lets that cell stretch across multiple columns horizontally.
  • rowspan: The rowspan attribute makes a cell span across multiple rows vertically in the same column.
